1.3.0
  • Added player colors (#15).

  • Added support for up to 30 players (#22).
  • Added portrait mode (#46). Note that landscape mode is not supported for more than 8 players.

  • "Manage Settings" → "Load Backup" works again (#65).
  • Tweaked the appearance on tablets.
  • Added launcher icons for xxhdpi and xxxhdpi resolutions.
  • Performance improvements.
